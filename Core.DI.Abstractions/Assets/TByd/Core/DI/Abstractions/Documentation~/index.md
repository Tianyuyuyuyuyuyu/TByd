# Core.DI.Abstractions

Unity依赖注入抽象层，提供框架无关的DI功能。

## 特性

- 🎯 框架无关的依赖注入抽象
- 🔄 支持多种生命周期管理
- 🎨 灵活的注入方式
- 🚀 异步初始化支持
- 📦 易于扩展的容器设计
