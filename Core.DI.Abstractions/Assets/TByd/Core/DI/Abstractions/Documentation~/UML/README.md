## 图例说明

### 关系类型

- `--|>` 继承关系
- `..>` 依赖关系
- `-->` 关联关系
- `--o` 聚合关系
- `--*` 组合关系

### 访问修饰符

- `+` public
- `-` private
- `#` protected
- `~` internal

### 构造类型

- `<<interface>>` 接口
- `<<abstract>>` 抽象类
- `<<static>>` 静态类
- `<<framework>>` 框架类
